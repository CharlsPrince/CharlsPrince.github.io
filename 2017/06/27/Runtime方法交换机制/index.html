一、运行时

所谓运行时，就是说程序在运行(或者在被执行时)的状态，运行时对于项目编程来说是一个必要重要的机制，能够帮我们处理很多未知的情况，比如说动态创建类、添加属性、方法等。

二、运用领域

  • 在程序运行时，动态创建一个类
  • 在程序运行时，动态为某类添加属性、方法，修改属性值、方法名称
  • 。。。

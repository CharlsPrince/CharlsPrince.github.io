准备工作

npm 安装完成了以后可以执行 npm -v 验证下是否安装成功

  1. 安装 plugman (全局安装)

npm install -g plugman

安装完可以执行 plugman -v 验证下是否安装成功

6. 注意项

基本到了这一步，一个简单的插件已经创建完了, 下面就是验证工作了。

不过，还有些地方应该注意的。

  • clobbers 中的 target 限制了你如何调用插件
  • config-file 中的内容会被配置到工程的config.xml中
